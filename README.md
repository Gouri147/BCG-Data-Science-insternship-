#  BCG Data Science Virtual Internship (Forage)

##  Project Overview
This project was completed as part of the BCG Data Science Virtual Internship on Forage. The objective was to analyze customer churn and identify the key factors influencing customer retention.

---

##  Problem Statement
PowerCo is facing customer churn and believes that price sensitivity is the main driver. The goal of this project was to validate this hypothesis using data analysis and machine learning.

---

##  Project Structure
- **data/** → Raw and processed datasets  
- **notebooks/** → Jupyter notebooks for EDA, feature engineering, and modeling  
- **documents/** → Executive summary and insights  
- **certificate/** → Internship completion certificate  

---

##  Key Steps Performed

### 1. Exploratory Data Analysis (EDA)
- Analyzed customer and pricing datasets  
- Studied distributions, missing values, and patterns  
- Identified no strong direct relationship between price and churn  

---

### 2. Feature Engineering
- Created features like:
  - Price sensitivity (6-month & yearly)
  - Price volatility  
  - Consumption behavior  
- Improved dataset for better modeling  

---

### 3. Model Building
- Built:
  - Logistic Regression  
  - Random Forest Classifier  
- Addressed class imbalance using `class_weight='balanced'`

---

##  Key Insights
- Price is **not the primary driver of churn**  
- **Customer acquisition channel (origin)** is the strongest predictor  
- Behavioral factors have more impact than pricing  

---

##  Business Recommendations
- Focus on **high-risk customer segments**  
- Improve **customer experience and engagement**  
- Use predictive models for **proactive churn prevention**  
- Apply **targeted pricing strategies** instead of blanket discounts  

---

##  Business Impact
- Enables early identification of churn-risk customers  
- Helps improve retention and customer lifetime value  
- Protects revenue by avoiding unnecessary discounts
